# \## 📂 Estrutura do Projeto

# 

# Atualmente, o repositório está organizado da seguinte forma:

# 

# \*   \*\*`/docs`\*\*: Contém a documentação de requisitos (Funcionais e Não Funcionais).

# \*   \*\*`/src`\*\*: Pasta destinada ao código-fonte da aplicação.

# \*   \*\*`/assets`\*\*: Arquivos de mídia, ícones e logotipos.

# \*   \*\*`/sql`\*\*: Scripts para criação e manutenção do banco de dados.
