\# Requisitos do Projeto - Clínica Médica



\### Requisitos Funcionais (RF)

\* \*\*RF01:\*\* O sistema deve permitir o cadastro de pacientes.

\* \*\*RF02:\*\* O sistema deve permitir o agendamento de consultas.

\* \*\*RF03:\*\* O médico deve poder registrar prontuários.



\### Requisitos Não Funcionais (RNF)

\* \*\*RNF01:\*\* O sistema deve ser acessível via navegador (Web).

\* \*\*RNF02:\*\* As senhas devem ser criptografadas (Segurança).

\* \*\*RNF03:\*\* O tempo de resposta das buscas deve ser inferior a 2 segundos.
