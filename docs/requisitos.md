\# 🏥 Projeto: Sistema de Gestão de Clínica Médica



Este documento descreve as funcionalidades e restrições técnicas do sistema desenvolvido para o projeto interdisciplinar.



---



\## **🛠 1. Requisitos Funcionais (RF)**

\*O que o sistema deve ser capaz de fazer.\*



**| ID | Funcionalidade | Descrição |**

|:---:|:--- |:--- |

**| \*\*RF01\*\* | \*\*Cadastro de Pacientes\*\* |** O sistema deve permitir registrar nome, CPF, data de nascimento e convênio. |

**| \*\*RF02\*\* | \*\*Agendamento\*\* |** O sistema deve permitir marcar, reagendar e cancelar consultas médicas. |

**| \*\*RF03\*\* | \*\*Prontuário Eletrônico\*\* |** O médico deve conseguir registrar observações, sintomas e diagnósticos na consulta. |

**| \*\*RF04\*\* | \*\*Gestão de Médicos\*\* |** O sistema deve gerenciar o cadastro de profissionais e suas especialidades. |

**| \*\*RF05\*\* | \*\*Controle de Acesso\*\* |** O sistema deve diferenciar o que a recepção e o que o médico podem visualizar. |

**| \*\*RF06\*\* | \*\*Emissão de Receitas\*\* |** O sistema deve gerar um documento formatado com a prescrição de medicamentos. |



---



**## ⚙️ 2. Requisitos Não Funcionais (RNF)**

\*As características de qualidade e segurança do sistema.\*



**| ID | Categoria | Descrição |**

|:---:|:--- |:--- |

**| \*\*RNF01\*\* | \*\*Segurança\*\* |** Todos os dados devem ser protegidos conforme a LGPD (Lei Geral de Proteção de Dados). |

**| \*\*RNF02\*\* | \*\*Desempenho\*\* |** As buscas por pacientes devem retornar resultados em menos de 2 segundos. |

**| \*\*RNF03\*\* | \*\*Disponibilidade\*\* |** O sistema deve ser acessível 24/7 para consultas de horários. |

**| \*\*RNF04\*\* | \*\*Interface\*\* |** O design deve ser limpo e intuitivo para facilitar o uso por funcionários da clínica. |

**| \*\*RNF05\*\* | \*\*Compatibilidade\*\* |** O sistema deve funcionar corretamente nos navegadores Chrome, Edge e Firefox. 

**| \*\*RNF06\*\* | \*\*Persistência\*\* |** O sistema deve utilizar um banco de dados relacional para garantir a integridade das informações. |
